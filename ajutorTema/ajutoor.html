Saptamana 11 - Laborator 10:  Prezentarea Temei 2 & OOB(out-of-band) data transmission intre TCP/IP client si server, folosind select()

https://www.ibm.com/docs/en/i/7.2?topic=concepts-out-band-data 

  • Examplu: un TCP server si un client. Clientii trimit mesaje serverului, intre care, la un anumit moment, niste OOB data este transmisa. Serverul foloseste select() sa primeasca standard data si OOB data. clientTcpOobSelect.c; serverTcpOobSelect.c;

  • Examplu: un TCP server si un client. Clientii trimit mesaje serverului, intre care, la un anumit moment, niste OOB data este transmisa. Serverul serveste semnalul SIGURG sa identifice OOB data. clientTcpOobSigurg.c; serverTcpOobSigurg.c;

  • Examplu: un TCP server si un client. Clientii trimit mesaje serverului, intre care, la un anumit moment, niste OOB data este transmisa. Serverul foloseste functia sockatmark() pentru a identifica OOB data. clientTcpOobSockat.c; serverTcpOobSockat.c;
